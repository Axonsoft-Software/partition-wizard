**Partition-Wizard** ist eine Windows-Anwendung zur Analyse, Verwaltung und Diagnose von Laufwerken, Datenträgern und Partitionen.

Diese README richtet sich an **Kunden und Endanwender** der Release-Version.

---

## Überblick

Partition-Wizard hilft dabei, angeschlossene Laufwerke übersichtlich darzustellen, Speicherinformationen zu prüfen und Datenträger besser zu verwalten.

Die Anwendung wurde für Windows entwickelt und bietet eine moderne Oberfläche mit klaren Anzeigen, Diagnosefunktionen und integrierter Update-Prüfung.

---

## Hauptfunktionen

- Anzeige verbundener Laufwerke und Datenträger
- Übersichtliche Darstellung von Speicherinformationen
- Erkennung von Partitionen und Dateisystemen
- Diagnosefunktionen für Laufwerke
- Unterstützung bei Analyse und Wiederherstellung
- Moderne grafische Benutzeroberfläche
- Integriertes Lizenzsystem
- Automatische Update-Prüfung
- Protokollierung wichtiger Ereignisse zur Fehleranalyse

## Installation

1. Lade die aktuelle Release-Version von Partition-Wizard herunter.
2. Entpacke die heruntergeladene Datei, falls sie als ZIP-Datei bereitgestellt wurde.
3. Starte die Anwendung über die Programmdatei.
4. Führe Partition-Wizard bei Bedarf als Administrator aus.
5. Aktiviere die Software mit deinem Lizenzschlüssel, falls erforderlich.

---

## Erste Schritte

Nach dem Start der Anwendung werden verfügbare Laufwerke und Datenträger angezeigt.

Empfohlene Reihenfolge:

1. Anwendung starten
2. Laufwerksübersicht prüfen
3. Gewünschtes Laufwerk auswählen
4. Diagnose oder Analyse starten
5. Hinweise und Meldungen sorgfältig lesen
6. Vor kritischen Aktionen ein Backup erstellen

---

## Lizenzaktivierung

Partition-Wizard verwendet ein Lizenzsystem zur Aktivierung der Software.

Für die Aktivierung wird in der Regel benötigt:

- gültiger Lizenzschlüssel
- aktive Internetverbindung
- korrekte Systemzeit
- Schreibrechte im Programm- oder Datenordner

Sollte die Aktivierung nicht funktionieren, prüfe bitte:

- ob der Lizenzschlüssel korrekt eingegeben wurde
- ob eine Internetverbindung besteht
- ob eine Firewall oder ein Antivirusprogramm die Verbindung blockiert
- ob die Anwendung mit ausreichenden Rechten gestartet wurde

---

## Updates

Partition-Wizard kann verfügbare Updates automatisch prüfen.

Der Updater kann:

- nach neuen Versionen suchen
- Update-Informationen anzeigen
- Update-Pakete herunterladen
- die Installation vorbereiten
- Fehler in einer Log-Datei speichern

Für Updates wird eine Internetverbindung benötigt.

---

## Sicherheitshinweise

Partition-Wizard arbeitet mit Informationen zu Laufwerken, Partitionen und Datenträgern.

Bitte beachte:

- Vor Änderungen an Laufwerken immer wichtige Daten sichern.
- Keine unbekannten oder falschen Datenträger auswählen.
- Kritische Aktionen nur durchführen, wenn du sicher bist.
- Bei externen Laufwerken vorher prüfen, ob das richtige Gerät verbunden ist.
- Während laufender Vorgänge das Programm nicht schließen.
- Während eines Updates das Gerät nicht ausschalten.

Der Entwickler übernimmt keine Haftung für Datenverlust durch falsche Bedienung, defekte Datenträger oder unsachgemäße Verwendung.

## Administratorrechte

Einige Funktionen können Administratorrechte benötigen.

Das betrifft unter anderem:

- erweiterte Laufwerksinformationen
- Diagnosefunktionen
- bestimmte Wiederherstellungsfunktionen
- Update-Vorgänge
- Zugriff auf geschützte Systembereiche

Falls eine Funktion nicht korrekt arbeitet, starte Partition-Wizard bitte erneut mit Administratorrechten.

## Fehlerbehebung

### Die Anwendung startet nicht

Prüfe bitte:

- ob Windows 10 oder Windows 11 verwendet wird
- ob die Datei vollständig heruntergeladen wurde
- ob ein Antivirusprogramm den Start blockiert
- ob die Anwendung aus einem schreibgeschützten Ordner gestartet wird

### Lizenz wird nicht erkannt

Prüfe bitte:

- ob der Lizenzschlüssel korrekt ist
- ob eine Internetverbindung besteht
- ob Datum und Uhrzeit auf dem PC korrekt sind
- ob die Anwendung Schreibrechte besitzt

### Update funktioniert nicht

Prüfe bitte:

- ob eine Internetverbindung besteht
- ob der Update-Server erreichbar ist
- ob Firewall oder Antivirus den Download blockieren
- ob genug Speicherplatz vorhanden ist
- ob Partition-Wizard als Administrator gestartet wurde

### Laufwerk wird nicht angezeigt

Prüfe bitte:

- ob das Laufwerk korrekt angeschlossen ist
- ob Windows das Laufwerk erkennt
- ob das Laufwerk beschädigt ist
- ob Administratorrechte benötigt werden
- ob das Laufwerk neu verbunden werden muss
